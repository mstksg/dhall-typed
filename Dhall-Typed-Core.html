data DType :: [DKind] -> DKind -> Type where Source #

Represents the possible types encountered in Dhall. A value of type

DType '[k1, k2, k3] k

Describes a type of kind k possibly containing free type variables of kind k1, k2, and k3.

Something of type DType '[] k is a type of kind k with no free variables.
